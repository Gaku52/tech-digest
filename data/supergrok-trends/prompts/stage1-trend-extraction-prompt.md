# 🎯 Stage 1: トレンド抽出プロンプト

## 目的
X（Twitter）上の技術トレンドから、エンジニアが知るべき重要な情報を抽出する。

## 実行指示

```
あなたは日本語テックニュースキュレーターです。2026年2月5日（または指定日）のX上の技術トレンドから、エンジニアにとって価値のある情報を抽出してください。

【重要】
- すべての出力は**日本語**で記載してください
- 英語のツイートは日本語に翻訳してください
- タイトル・要約・重要ポイントはすべて日本語で記載
- 技術用語は適切に日本語化してください
- 固有名詞（製品名、企業名）は原語のまま使用してください

【検索条件】
- 検索日: 2026-02-05（未来日の場合は前日まで）
- 対象言語: 英語・日本語
- エンゲージメント: いいね+リツイート合計100以上
- 対象分野: AI、Blockchain、Engineering、Developer Tools、Infrastructure

【出力要件】
以下のJSON形式で、優先度別に15本のトレンドを抽出してください:

```json
{
  "date": "2026-02-05",
  "trends": [
    {
      "id": "trend-001",
      "title": "記事タイトル（日本語、60文字以内）",
      "category": "AI | Blockchain | Engineering",
      "priority": "critical | high | medium | low",
      "tags": ["タグ1", "タグ2", "タグ3"],
      "excerpt": "記事の要約（100-150文字）",
      "engagement": 15000,
      "sources": [
        "https://x.com/username/status/123456789"
      ],
      "keyPoints": [
        "重要ポイント1（具体的な数値・事実を含む）",
        "重要ポイント2（具体的な数値・事実を含む）",
        "重要ポイント3（具体的な数値・事実を含む）"
      ],
      "technicalKeywords": [
        "技術キーワード1",
        "技術キーワード2",
        "技術キーワード3"
      ]
    }
  ]
}
```

【優先度の定義】
- **Critical (3本)**: 業界に即座に影響を与える重大発表（新技術リリース、破壊的イノベーション）
- **High (3本)**: 1-2週間以内に対応を検討すべき重要情報（メジャーアップデート、セキュリティ問題）
- **Medium (3本)**: 1-2ヶ月以内に把握すべき有用情報（新機能、ベストプラクティス）
- **Low (6本)**: 知っておくと良い情報（マイナーアップデート、コミュニティ動向）

【重要な注意事項】
1. **具体性を重視**: keyPointsには「すごい」「画期的」ではなく、具体的な数値・スペック・事実を記載
2. **情報源の明確化**: 必ず元ツイートのURLを記載
3. **エンゲージメント順**: 各優先度内で、エンゲージメントが高い順に並べる
4. **重複排除**: 同じトピックの重複は避け、最もエンゲージメントが高いものを選択
5. **technicalKeywordsの重要性**: 第2段階で深掘りする際の検索キーワードとして使用

【出力形式】
- JSON形式のみ
- コメント不要
- 15本すべてを出力（Critical 3 + High 3 + Medium 3 + Low 6 = 15）
```

## 出力ファイル名

`data/supergrok-trends/YYYY-MM-DD/stage1-trends.json`

例: `data/supergrok-trends/2026-02-05/stage1-trends.json`

## 次のステージへの引き継ぎ

このJSONファイルの`technicalKeywords`と`sources`を使って、Stage 2で技術的詳細を深掘りします。
